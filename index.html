ELEC 327 Final Project

By John Reko, Peter Humphreys, and Sean Hamilton

Overview

For our final project, we created an LED lighting system using an MSP430G2553. The user plays audio through the aux input, and the LEDs light up in response to the volume levels of different frequency ranges in the audio. To learn more about the details of this project, you can view our project report below.
